# Seed data review — Pekka + Liina 90v

Edit this file directly, then tell me when you're done and I'll regenerate
`src/lib/seedData.ts` from it and push to Vercel.

How to edit:
- Fix names, responsible people, amounts, or notes directly in the table cells.
- Delete a whole row to remove that item.
- Add a new row in the same `| Name | Responsible | Amount | Note | ⚠️ |` format to add an item.
- Put an `x` in the **⚠️** column if the item still needs someone to fill in
  details (shown in the app as a warning icon). Leave it empty otherwise.
- You can rename a `###` group heading, add a new group, or delete one
  entirely (delete its heading + table).
- Don't rename the `##` section headings (`todo`, `misa`, `aineet`) — those
  map to the app's fixed sections.
